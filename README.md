# ClaudeWorksHub Plugin Marketplace

Claude Code plugin marketplace maintained by [ClaudeWorksHub](https://github.com/ClaudeWorksHub).

## Available Plugins / 可用插件

| Plugin | Description | Version |
|--------|-------------|---------|
| [huashanlunjian](https://github.com/sunfjun/huashanlunjian) | 华山论剑 — Multi-AI role debate for high-quality proposals | 1.1.0 |

## Installation / 安装

```bash
# 1. Add marketplace / 添加市场
/plugin marketplace add ClaudeWorksHub/claude-plugins

# 2. Install a plugin / 安装插件
/plugin install huashanlunjian@claudeworkshub

# 3. Activate / 激活
/reload-plugins
```

## Update / 更新

```bash
/plugin marketplace update claudeworkshub
/plugin install huashanlunjian@claudeworkshub
/reload-plugins
```
